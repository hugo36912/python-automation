juniper to cisco vpn:
https://www.networkstraining.com/site-to-site-ipsec-vpn-between-cisco-router-and-juniper-security-gateway/#:~:text=The%20idea%20is%20simple%3A%20configure,the%20same%20on%20both%20routers.
isakmp=ike,hash =authentication-algorith,gateway=isakmp profile,transform-set=ipsec proposol
crypto isakmp policy 3
 encr aes 256
 hash sha256
 authentication pre-share
 group 20
crypto isakmp key 123456 address 3.3.3.3
!
crypto ipsec transform-set 340 esp-3des esp-md5-hmac
mode tunnel
!
crypto ipsec profile 340
 set transform-set 340
!

interface Tunnel1
 ip address 3.3.2.5 255.255.255.248
 tunnel source 3.3.3.4
 tunnel mode ipsec ipv4
 tunnel destination 3.3.3.3
 tunnel protection ipsec profile 340
